Tennis: VR Game

This Unity game is made for Meta Quest 3. It simulating real-life tennis
and supports single player mode with a computer opponent.

Handcontrollers are used, with grab buttons used to grab the racket and the ball.
The red trajectory line predicts the line on which the ball should fall.
